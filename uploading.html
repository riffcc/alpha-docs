Uploading to Riff.CC

Uploading to Riff is one of the coolest ways to contribute to the site. Just for considering it, we want to thank you! Make sure everything is legally free before you begin - it should be clearly marked as public domain or Creative Commons.

What software do I need to upload to Riff.CC?

You can use any client you're comfortable with, but we suggest using BiglyBT.

Grab a copy of BiglyBT @ biglybt.com and install it. If you run it as Administrator (optional), it will be a little bit easier to associate with torrent files. Open it up, click Help, Join Beta Program and follow the prompts to switch to the beta version.

Open Bigly again, then go to Tools -> Options. Click Queue. Then ensure "Don't start any more torrents when upload/download limit is reached" is unchecked. Then click -> Seeding -> Ignore Rules and ensure that "Ignore torrents with 0 peers" is UNSET. Hit Save and close the window. You only have to do this once!

How do I upload music?

Once you've set up BiglyBT, download the content you want to upload and make sure it's in a nice safe place. Navigate to Riff.CC and click Publish -> Music (the Up icon on the left side of your screen). Copy the "announce URL" - it will look like https://u.riff.cc/announce/abzd66b4eb8x1b391ee7ez8554f3674e.

Open BiglyBT and click File -> New Torrent. Select "Use an external Tracker" and paste in the Announce URL from Riff.CC. Make sure to set the "Source" field to "Riff.CC". Add your files/folders. Hit next, ensuring "Private torrent" and "Open the torrent for seeding when done" are both selected. Hit finish and you're done!

Once that's done, you have a torrent file ready to upload. Back on Riff.CC, click the first "Choose File" and select the file you just created. Select an appropriate Type, fill out the description with where you found the music and a rough description of it, and ideally which licence it's available under. Then hit Submit! You've just uploaded your first thing!

Once you've uploaded everything you plan to upload for the day, simply click File -> Restart BiglyBT.

How do I upload other kinds of content?

The process is exactly the same! Just pick what you want to publish instead of Music, and follow the prompts. If you're uploading TV or Movies, you will need to make sure a TMDB entry exists for the content you're uploading. If one doesn't exist - create one!
